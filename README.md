## Description

Have you ever wanted to enhance your gaming experience by understanding NPC dialogues in both your native language and English? With Dialog with English, NPC dialogues will feature dual subtitles, including the language you want to learn

## Features

- **Dual-Language Display**: NPC dialogues will appear in both your selected language and English.
- **Enhanced Learning & Immersion**: Great for players who want to improve their English comprehension while enjoying the game.
- **Support In-game Dialog**:
  - ✅ **Overhead Dialog**
  - ✅ **Speech Dialog**
  - ✅ **Dialog Sequence**

## Supported Language Combination

Since Kingdom Come: Deliverance 2's language files are divided into separate language packs, including **Simplified Chinese**, **Traditional Chinese**, **Japanese**, **Korean**, and **the default Latin-based** language pack.
- **Latin-Based Font Pack** – Does not include any Asian language characters.
- **Asian Font Pack (Simplified Chinese, Traditional Chinese, Japanese, Korean)** – Includes fonts for their respective languages along with common Latin letters.

The game determines which language pack to load based on the language setting in the game options. Therefore, if the language chosen for dual subtitles is not included in the loaded language pack, it will not be displayed in the game. As a result, the following combinations are recommended:
- Game Language: Latin-based (e.g., English) → Paired with another Latin-based language (e.g., French) ✅ No issues.
- Game Language: Asian-based (e.g., Chinese) → Paired with English ✅ No issues.
- Game Language: Asian-based (e.g., Chinese) → Paired with Russian ⚠ Some missing characters.
- Game Language: Latin-based (e.g., English) → Paired with an Asian-based language (e.g., Chinese) ❌ Most Asian characters will be missing.

## How to use

1. **Download the mod** from the _Files_ section or the GitHub release page.
2. **Launch the tool** and generate your mod by selecting your game language and the additional language you want to display.
3. **Extract the contents of _Dual Dialog.zip_** into your game’s `Mods` folder.
4. **Launch the game** and enjoy NPC dialogues in both your language and English!

This mod does not modify game levels or interfere with quest progressions.
